## 中文版

### 优点

- 3分钟轻松搭建wurst环境
- 懒人bat配置，轻松运行演示地图
- 代码可结合lua引擎，内置japi，ydwe japi，dzapi，11api等均可兼容。

### 使用须知：

1..vscode文件夹，修改叫runmap.bat的文件 
填写war3和ydwe路径【视频里有】

2.安装vscode插件，直接搜索就能安装了
【chinese】 【wurst】【lua debug】【Tasks】

3.点左下角【编译并运行】，然后屏幕中间弹出菜单，选择你要编译的地图，等一会就好了。
